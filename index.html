Auto Register

다운로드

  • 최신 버전(2021/07/26): Auto Register v1.2
  • 기존 사용자는 exe 파일만 교체하시면 기존 로그인 설정이 유지됩니다.

프로그램 정보

  • 동작환경 : Windows 10 64bit
  • 문의 : soohyun.dev@gmail.com

사용 방법

  1. 프로그램 압축파일을 다운로드 받습니다.
  2. 압축해제후 userDate 파일을 메모장으로 열어줍니다.
  3. {}안에 ID, PW, 팝업 비활성화 여부를 입력 후 저장합니다.
  4. Auto_register.exe 실행합니다.

부가 기능

강의 입장 확인

  • 해당하는 날짜의 강의가 여러 개라면 입장할 강의를 확인하는 팝업을 노출합니다.
  • userDate에서 팝업을 비활성화할 수 있습니다.
